html 태그 5가지

  • '<img>' : 이미지를 넣을 때 쓰는 태그
  • '<i>' : 이태릭체로 출력하는 태그
  • '<xmp>' : 태그를 그대로 출력할 때 쓰는 태그
  • '<b>' : 볼드체로 출력하는 태그
  • '<br>' : 줄을 바꾸는 태그

클라이언트와 서버

  • 클라이언트는 서비스를 사용하는 사용자이다.
  • 서버는 일반적으로 클라이언트에게 네트워크를 통해 서비스를 제공하는 시스템이다.

CSS는 왜 필요할까

  • CSS는 HTML만으로 하기 어려운 작업을 손쉽게 하게 해주며 사이트의 스타일을 빠르고 쉽게 제작할 수 있다.
